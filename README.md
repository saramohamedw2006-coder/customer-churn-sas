# Customer Churn Prediction using SAS

## Project Overview
This project predicts customer churn using SAS.

## Dataset
Customer churn dataset containing:
- Age
- Tenure
- Usage Frequency
- Support Calls
- Payment Delay
- Total Spend
- Subscription Type
- Contract Length
- Churn

## Steps
1. Data Import
2. EDA
3. Data Cleaning
4. Feature Engineering
5. Standardization
6. Logistic Regression
7. Decision Tree
8. Model Evaluation

## Models Used
- Logistic Regression
- Decision Tree

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Key Findings
- Payment delays strongly affect churn
- Support call rate is important
- Logistic regression performed better/worse than tree model
